웹 사용 방법

GSG 학습 플랫폼의 모든 기능을 한눈에 파악하고 효율적으로 활용하세요.

목차

학습 성과 보상 커뮤니티

학습

개별 문제 학습

  • 과목 → 대/중/소분류 → 개별 문항 순서로 선택하여 문제 풀이
  • 정답 시 경험치/포인트 획득 (일일 한도: 경험치 2,000 / 포인트 1,000)
  • 72시간 쿨다운: 동일 문항 재풀이 시 보상 제한
  • 부스터 사용 시 포인트/일일한도 2배 적용
  • 즐겨찾기/오답 관리: 중요 문항과 틀린 문제 자동 수집
개별 학습

모의고사

  • 범위 설정: 과목/분류별로 학습 범위 선택
  • 난이도 조절: 하트 1~5개로 난이도 범위 설정
  • 최소 10문항부터 모의고사 생성 가능
  • 하루 1회 보상: 최종 제출 후 경험치/포인트 지급
  • 실시간 문항 수: 설정한 범위에서 검색 가능한 문항 수 표시
모의고사

즐겨찾기/오답

  • 즐겨찾기: 중요하다고 표시한 문항들을 모아서 복습
  • 오답 관리: 틀린 문제들을 자동으로 수집하여 재학습
  • 즉시 이동: 해당 문항으로 바로 이동하여 풀이 가능
  • 개별 학습 연동: 즐겨찾기/오답에서 바로 문제 풀이
즐겨찾기

성과

학습 통계

  • 진행률 차트: 일자별 고유 문항 풀이 비율 시각화
  • 정답률 분석: 과목/분류별 정답률 통계
  • 일별 풀이 현황: 매일 풀이한 문항 수 추이
  • ECharts 기반: 인터랙티브 차트로 데이터 탐색
  • 실시간 업데이트: 학습 진행에 따라 통계 자동 갱신
학습 통계

성취도/리더보드

  • 레벨 시스템: 경험치 기반 레벨업 (1레벨업당 +1코인)
  • 배지/업적: 9가지 업적 달성 시 코인 보상
  • 챌린지 진행: 일/주/월별 문제 풀이 목표 달성
  • 리더보드: 누적 경험치/레벨 기준 전체 순위
  • 진행률 표시: 각 챌린지별 달성률 실시간 확인
성취도

개인화 추천

  • 난이도 추천: 최근 성과를 반영한 적정 난이도 제안
  • 취약 영역 보완: 정답률이 낮은 분야 우선 추천
  • 전역 오답률 TOP: 전체 사용자 중 가장 많이 틀린 문항
  • 학습 패턴 분석: 개인별 학습 성향 파악
개인화 추천

보상

지갑

  • 포인트/코인 관리: 보유 포인트와 코인 잔액 확인
  • 아이템 관리: 구매한 상품들의 사용/환불 처리
  • 부스터 상태: 포인트/일일한도 2배 효과 확인
  • 거래 내역: 포인트/코인 획득/사용 기록
  • 90% 환불: 미사용 아이템 구매가의 90% 환불
지갑

포인트 상점

  • 교환권/특별 아이템: 포인트로 다양한 상품 구매
  • 부스터 아이템: 포인트/일일한도 2배 효과 (48시간)
  • 코인 구매: 포인트로 코인 교환 가능
  • 레벨 제한: 일부 상품은 최소 레벨 필요
  • 구매 내역: 과거 구매한 상품들 확인
포인트 상점

로또

  • 즉시 추첨: 6개 번호 선택 후 즉시 당첨 결과 확인
  • 1코인 1회: 1코인으로 로또 1장 구매
  • 5등급 당첨: 1등~5등까지 당첨 등급 확인
  • 구매 내역: 과거 구매한 로또 티켓들 확인
  • 개인 통계: 본인의 당첨 현황 통계
로또

코인 거래소

  • 매수/매도 주문: 포인트로 코인 매수, 코인으로 포인트 매도
  • 오더북: 실시간 매수/매도 호가 현황
  • 체결 내역: 거래 성사된 내역 확인
  • 정산 시스템: 거래 완료 후 포인트/코인 정산
  • 매시 정각 거래: 매시 정각에 주문 매칭 실행
거래소

커뮤니티

스터디 그룹

  • 1인 1그룹: 사용자당 한 개 그룹만 소속 가능
  • 그룹 생성/참여: 새 그룹 생성 또는 6자리 초대 코드로 참여
  • 주간 챌린지: 그룹원 수 × 50문제 목표 달성
  • 그룹 내 순위: 이번 주 문제 풀이 순위
  • 코인 선물: 같은 그룹원끼리만 코인 선물 가능
  • 실시간 채팅: 그룹원들과 실시간 대화
  • 그룹 관리: 그룹장은 정보 수정/해체, 그룹원은 탈퇴 가능
스터디 그룹

질문과 답변(Q&A)

  • 과목/분류/문항까지 선택하여 구체적인 질문 작성
  • 누구나 답변: 모든 사용자가 질문에 답변 가능
  • 답변 채택: 질문 작성자가 가장 좋은 답변 선택
  • 검색/필터: 과목, 분류, 상태별 질문 검색
  • 정렬 기능: 최신순, 인기순, 답변순으로 정렬
  • 활동 랭킹: 질문/답변/채택 수 기준 사용자 순위
  • 본인 답변만 수정: 작성자는 자신의 답변만 수정 가능
Q&A 시스템

피드백

  • 카테고리 분류: 오류/개선/기능 요청으로 분류
  • 상세 설명: 문제 상황과 개선 방안 상세 작성
  • 관리자 응답: 관리자만 답변/상태 변경 가능
  • 상태 관리: 접수/응답/해결 상태로 진행 상황 표시
  • 검색/필터: 카테고리, 상태별 피드백 검색
  • 인증 사용자만: 로그인한 사용자만 피드백 작성 가능
피드백
