👋 송현여고 수학 학습 시스템

개념별 기출문제로 실력을 키워보세요

📝 회원 정보 입력

학습 현황을 정확히 분석하기 위해 기본 정보를 입력해주세요.

🎯 학습 시스템 특징

  • 개념별 학습: 수학의 각 개념별로 체계적으로 정리된 기출문제
  • 실시간 피드백: 문제 풀이 후 즉시 정답 여부 확인
  • 개인별 분석: 학습 현황과 정답률을 한눈에 확인
  • 게이미피케이션: XP, 배지, 스트릭으로 학습 동기부여
  • 즐겨찾기: 중요한 문제를 저장하여 나중에 복습

📊 초기 설정 가이드

1

기본 정보 입력

학년, 반, 번호, 이름을 정확히 입력해주세요.

2

수학 수준 설정

현재 수학 성적 수준을 선택하면 맞춤형 문제를 추천해드립니다.

3

학습 목표 설정

학습 목표를 설정하면 개인화된 학습 계획을 제공합니다.

4

학습 시작

설정이 완료되면 바로 학습을 시작할 수 있습니다.

🔐 안내 및 유의사항

  • 로그인 방식: Google 계정으로 로그인합니다. 브라우저 팝업/리다이렉트가 차단되지 않았는지 확인해주세요.
  • 개인정보 최소 수집: 학습 식별과 통계를 위해 학년/반/번호/이름만 수집합니다.
  • 코인 시스템: 최초 가입 시 기본 코인이 지급되며 매일 일일 로그인 보상이 자동 지급됩니다.
  • 데이터 저장소: 핵심 데이터는 Firestore에 안전하게 저장됩니다. 네트워크 불안정 시 재시도 후 진행해주세요.
  • 기존 사용자: 같은 Google 계정으로 로그인하면 기존 프로필과 병합/갱신됩니다.
