웹 사용 방법

GSG 학습 플랫폼의 학습·성과·보상·커뮤니티 기능을 빠르게 이해하고 시작하세요.

목차

시작하기 학습 성과 보상 커뮤니티 유의사항

시작하기

  • 접속: 개발/로컬 환경에서는 http://localhost 같은 http/https 주소로 접속하세요. 파일 열기(file://)에서는 인증/데이터 기능이 동작하지 않습니다.
  • 로그인: 메인에서 Google 로그인 후 사용합니다. 최초 로그인 시 업적/코인이 적용될 수 있습니다.
  • 프로필: 로그인 직후 필요한 경우 프로필 입력 페이지로 이동하여 학년/반/번호/이름/별명/전화번호를 저장합니다.
메인으로 이동

학습

개별 문제 학습

  • 과목 → 대/중/소분류 → 개별 문항을 선택해 풀이합니다.
  • 정답 제출 시 경험치/포인트가 지급되며, 동일 문항 보상은 72시간 쿨다운이 적용됩니다.
  • 오늘 획득 한도(경험치 2,000 / 포인트 1,000) 내에서 적립됩니다. 부스터 사용 시 포인트/일일한도가 2배로 적용됩니다.
  • 문항은 즐겨찾기로 관리하거나 오답은 자동 수집됩니다.
개별 학습

모의고사

  • 범위(과목/분류)와 난이도(하트)를 설정해 문항 풀을 구성한 뒤 시험을 생성합니다.
  • 최종 제출 후 결과 요약을 확인하고, 하루 1회 보상이 지급됩니다.
모의고사

즐겨찾기/오답

  • 즐겨찾은 문항과 오답 문항을 모아보고, 해당 문항으로 바로 이동해 복습합니다.
즐겨찾기

성과

학습 통계

  • 진행률(일자별 고유 문항 비율), 정답률(과목/분류별), 일자별 풀이 개수를 ECharts로 시각화합니다.
학습 통계

성취도/리더보드

  • 레벨, 배지/업적, 일/주/월 챌린지 진행 상황을 확인합니다.
  • 누적 경험치/레벨 기준 리더보드를 제공합니다.
성취도

개인화 추천

  • 최근 성과를 반영한 난이도 추천, 취약 영역 보완, 전역 오답률 TOP 문항을 제공합니다.
개인화 추천

보상

지갑

  • 보유 포인트/코인과 아이템을 확인합니다.
  • 아이템 사용/환불이 가능하며, 부스터(포인트/일일한도 2배) 상태를 확인할 수 있습니다.
지갑

포인트 상점

  • 포인트로 교환권/특별 아이템(부스터, 코인 등)을 구매합니다.
포인트 상점

로또

  • 번호 6개 선택 후 즉시 추첨 결과를 확인합니다. 구매 내역/통계를 제공합니다.
로또

코인 거래소

  • 매수/매도 주문을 제출하고, 오더북·체결·정산 현황을 확인합니다. 차트는 24시간 거래 기준입니다.
거래소

커뮤니티

스터디 그룹

  • 사용자당 한 개 그룹만 소속 가능. 그룹 생성 또는 초대 코드로 참여합니다.
  • 그룹장: 정보 수정/해체, 그룹원: 탈퇴 가능. 해체 시 그룹 정보/소속이 모두 삭제됩니다.
  • 주간 챌린지(그룹원 수 × 50 문제) 진행도를 확인하고, 그룹 내 순위를 확인합니다.
  • 같은 그룹끼리만 코인 선물 가능, 실시간 채팅 지원, 닉네임으로 표시됩니다.
스터디 그룹

질문과 답변(Q&A)

  • 과목/분류/문항까지 선택하여 질문을 올리고, 누구나 답변할 수 있습니다.
  • 작성자는 본인 답변만 수정 가능하며, 질문 작성자가 답변 채택을 할 수 있습니다.
  • 검색/필터/정렬과 인기 질문 목록, 사용자 활동 랭킹(질문/답변/채택 수) 제공.
Q&A 시스템

피드백

  • 카테고리(오류/개선/기능 요청)와 설명을 작성합니다. 작성자는 인증 사용자만 가능합니다.
  • 관리자만 답변/상태(접수/응답/해결) 변경이 가능합니다.
  • 검색/필터로 피드백과 응답을 확인합니다.
피드백

유의사항

  • 일부 기능은 인증이 필요합니다. 인증 상태에 따라 메뉴가 제한될 수 있습니다.
  • 데이터는 Firebase에 저장되며, 네트워크/권한 문제 시 토스트 메시지가 표시됩니다.
  • 문항 데이터는 questions.json과 이미지(mun/*.png)를 사용합니다.
  • 브라우저/네트워크 환경에 따라 일부 차트/실시간 기능이 지연될 수 있습니다.
