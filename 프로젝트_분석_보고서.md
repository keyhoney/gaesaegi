# 송현여고 수학 학습 시스템 - 상세 기능 분석 보고서

## 📋 프로젝트 개요

이 프로젝트는 **송현여자고등학교 수학 학습 시스템**으로, 학생들이 수학 기출문제를 체계적으로 학습할 수 있는 웹 애플리케이션입니다. Firebase를 기반으로 한 실시간 데이터베이스와 인증 시스템을 활용하여 개인화된 학습 경험을 제공합니다.

## 🏗️ 전체 아키텍처

### 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **PWA**: Service Worker, Web App Manifest
- **UI/UX**: 반응형 디자인, 모바일 최적화

### 핵심 기능
1. **구글 로그인 시스템**
2. **수학 기출문제 학습**
3. **게임화 시스템 (레벨, 경험치, 배지)**
4. **코인 시스템 (거래소, 로또, 선물)**
5. **개인화된 학습 추천**
6. **모의고사 및 성적 분석**
